# Inmobiliaria Web Documentation

Documentation for the Inmobiliaria Web real estate platform.

## About

This documentation covers the full-stack real estate platform including:

- Property listings and search functionality
- User authentication with Better Auth
- Interactive maps with Google Maps and Leaflet
- Admin panel for property and user management
- Complete API reference
- Development setup and deployment guides

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview the documentation locally:

```bash
npm i -g mint
```

Run the following command at the root of your documentation:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.

## Documentation Structure

- **Get Started** - Introduction, quickstart, and installation guides
- **Core Features** - Property listings, search, authentication, favorites, and management
- **User Guide** - End-user guides for browsing, creating listings, and admin panel
- **Components** - Component documentation with props and usage examples
- **API Reference** - Complete API endpoint documentation
- **Development** - Environment setup, architecture, and integrations

## Publishing Changes

Changes are deployed to production automatically after pushing to the default branch.
